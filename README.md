# 🏭 Dashboard Giám Sát Dây Chuyền Sản Xuất LEGO

Ứng dụng Streamlit tích hợp 3 vấn đề:
- **Vấn đề 1**: Tối ưu chất lượng sản phẩm ép phun
- **Vấn đề 2**: Bảo trì dự đoán cho hệ thống khuôn và trục vít
- **Vấn đề 3**: Phân loại lỗi ngoại quan bằng hình ảnh

## 🚀 Chạy local

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🌐 Deploy

Ứng dụng được deploy trên [Streamlit Community Cloud](https://streamlit.io/cloud).

## 📦 Yêu cầu

- Python 3.9+
- Streamlit
- Pandas
- Pillow
